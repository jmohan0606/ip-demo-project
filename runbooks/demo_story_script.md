# Demo Story Script

## Demo Title

iPerform Insights & Coaching — Enterprise Wealth Management AI Demo

## Persona Setup

Start with:

```text
Persona: Advisor
Scope: Advisor ADV0001
Time Period: YTD
```

## Story

1. The advisor opens the dashboard and sees performance insights.
2. The system prepares local signals:
   - Features
   - Embeddings
   - Predictions
   - Opportunities
   - Recommendations
3. The dashboard shows explainable insights and coaching cards.
4. The advisor drills into Advisor 360.
5. The advisor reviews recommendations.
6. The advisor accepts a recommendation.
7. The feedback learning loop creates:
   - Feedback event
   - Outcome event
   - Learning signal
   - Memory update
8. The advisor asks the AI Assistant:
   - “Why is my revenue low and what should I do next?”
9. The assistant answers using:
   - Context memory
   - Knowledge/RAG
   - Insights
   - Predictions
   - Opportunities
   - Recommendations

## Key Message

This is not just a dashboard. It is a graph-grounded, memory-enabled, feedback-learning advisor coaching system.
