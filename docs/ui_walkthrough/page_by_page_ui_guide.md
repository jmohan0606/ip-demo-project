# Streamlit UI Page-by-Page Guide

## Executive Dashboard

Main landing dashboard with insight cards, coaching cards, evidence, reasoning, and next best actions.

## End-to-End Demo Run

Runs the complete demo pipeline locally.

## Advisor 360

Advisor-specific view of features, predictions, opportunities, and recommendations.

## AGP Goals & Coaching

Advisor Growth Program goal tracking, KPI status, off-track metrics, and coaching signals.

## Opportunities

Detects and displays ranked advisor opportunities.

## Recommendations

Generates playbook-supported, compliance-checked recommendations.

## Feedback Learning

Captures accept/reject/complete feedback and generates learning signals.

## Feature Store

Materializes and inspects local SQLite feature vectors.

## Graph Embeddings & Similarity

Builds local graph embeddings and peer/household similarity.

## Predictions

Runs sklearn prediction models and displays model metadata.

## Knowledge Management

Ingests documents into Chroma and links document metadata to graph access layer.

## Context Memory

Creates and retrieves firm/division/region/market/advisor memory.

## Data Ingestion & Sync

Runs resumable CSV upload and graph upsert workflow.

## AI Assistant Chat

Separate chat page using memory, RAG, insights, predictions, opportunities and recommendations.

## Graph Access Status

Validates active graph path: MCP, REST, or Mock.

## Runtime Status

Displays application runtime health.
